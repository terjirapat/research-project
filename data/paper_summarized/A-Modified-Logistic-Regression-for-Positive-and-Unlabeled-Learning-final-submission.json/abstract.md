# A-Modified-Logistic-Regression-for-Positive-and-Unlabeled-Learning-final-submission — Abstract

**Research Problem:**
The paper addresses the positive and unlabeled (PU) learning problem, a semi-supervised binary classification challenge where only a subset of positive samples is known, while the rest of the samples, including both positive and negative ones, are unknown.

**Proposed Approach:**
The authors propose a modified logistic regression learner with a variable upper bound to better solve the PU learning problem. This modification is built on an existing probabilistic PU learning algorithm.

**Experimental Setup:**
The proposed method is tested on both simulated data and a real-world image classification problem using the MNIST dataset.

**Key Results:**
The new logistic regression learner with the variable upper bound significantly improves classification performance when applied to the MNIST dataset.

**Main Contribution:**
The paper introduces a modified logistic regression learner with a variable upper bound, which provides a better theoretical solution for the PU learning problem and demonstrates improved performance on both simulated and real-world datasets.
